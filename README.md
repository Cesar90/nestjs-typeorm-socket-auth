#Teslo API

1. Clone project
2. `npm install `
3. Clone the `.env.template` file and rename it to `.env`
4. Change of variables of env
5. Up the database

```
docker-compose up -d
```

6. Up `npm run start:dev`

7. Execute SEED

```
http://localhost:3000/api/seed
```
